for-each - Probleme

  • Böse Fehler bei Erweiterung von Standard-Objekten
  • Beispiel: Array-Erweiterungen in MooTools
// MooTools dynamisch laden
            demo.loadMooTools(function () {
                var array = [5,4,3,2,1,0];
                for (var i in array) {
                    console.log(i);
                }
                // 0, 1, 2, 3, 4, 5, $family, $constructor, pop, push ...
            });

>Workaround: Mit hasOwnProperty prüfen

for (var i in array) {
                if (array.hasOwnProperty(i)) {
                    console.log(i);
                }
            }
            // 0, 1, 2, 3, 4, 5
